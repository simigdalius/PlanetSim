# 🪐 Planet Simulation

![Pygame](https://img.shields.io/badge/Pygame-Library-green?style=for-the-badge)

Μια 2D προσομοίωση του ηλιακού μας συστήματος (Ήλιος και εσωτερικοί πλανήτες) βασισμένη στη Νευτώνεια Φυσική, γραμμένη σε Python με χρήση της βιβλιοθήκης Pygame. 

Αυτό το project δημιουργήθηκε ακολουθώντας το tutorial του [Tech with Tim](https://www.youtube.com/c/TechWithTim).


##  Σχετικά με την Προσομοίωση

Ο κώδικας προσομοιώνει τις τροχιές του **Ερμή, της Αφροδίτης, της Γης και του Άρη** γύρω από τον **Ήλιο**. 
Για τον υπολογισμό της κίνησης και της έλξης μεταξύ των ουράνιων σωμάτων, το πρόγραμμα χρησιμοποιεί τον Νόμο της Παγκόσμιας Έλξης του Νεύτωνα:

$$F = G \frac{m_1 m_2}{r^2}$$

Όπου:
* **F**: Η δύναμη της βαρύτητας
* **G**: Η σταθερά της βαρύτητας ($6.67428 \times 10^{-11}$)
* **m1, m2**: Οι μάζες των δύο σωμάτων
* **r**: Η απόσταση μεταξύ των κέντρων τους

##  Χαρακτηριστικά

* **Πραγματικές Τιμές:** Χρήση πραγματικών μαζών, αποστάσεων (σε Αστρονομικές Μονάδες - AU) και αρχικών ταχυτήτων για τους πλανήτες.
* **Δυναμικός Υπολογισμός:** Η βαρυτική έλξη υπολογίζεται σε πραγματικό χρόνο (60 FPS) μεταξύ **όλων** των σωμάτων, όχι μόνο με τον Ήλιο.
* **Οπτικοποίηση Τροχιάς:** Το πρόγραμμα σχεδιάζει δυναμικά τη γραμμή της τροχιάς (orbit trail) που ακολουθεί ο κάθε πλανήτης.
* **Live Δεδομένα:** Εμφάνιση της τρέχουσας απόστασης του κάθε πλανήτη από τον Ήλιο σε χιλιόμετρα.
